Risiko is a web based tool that models impacts of different hazard events on population or infrastructure. It is Open Source Software, and we encourage you to build new applications using its components and the resources it provides. This page provides links to the source code, the documentation and the online demo server.

What is Risiko made out of?

Risiko is built using several open source projects, each with its own community. If you are interested in contributing new features to the Risiko, we encourage you to do so by contributing to one of the projects on which it is built:

  • GeoNode - The spatial data infrastructure that integrates robust social and cartographic tools
  • GDAL/OGR - The translator library for raster and vector geospatial data formats
  • SciPy/Numpy - The open-source library for mathematics, science, and engineering

Who started Risiko?

Risiko - Risk in a Box started with a parnership from the Australia Indonesia Facility from Disaster Reduction (AIFDR), the Global Facility For Disaster Recovery and Reduction (GFDRR) and The World Bank. The development team is distributed around the world but the first version of Risiko was written in Jakarta, Indonesia with support from our local partners at BPPT.
